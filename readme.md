# MIPS Simulator in C

20-2학기 컴퓨터구조 기말 팀프로젝트 MIPS 시뮬레이터 구현

Repl.it 링크: https://repl.it/@yongjin0802/mips-simulator-team-0x03#main.c

## 팀원

- 조용진 (팀장)
- 김상우
- 오석진

## 실행법

### 바이너리 읽기

```
l 1.bin
```

### 프로그램 전체 실행

```
g
```

### 프로그램 한 단계 실행

```
s
```

### 메모리 보기

```
m 0x400000 0x400010
```

### 메모리 설정

```
sm 0x400000 0x400010
```

### 레지스터 보기

```
r
```

### 레지스터 설정

```
sr 10 0x4000
```

### 프로그램 종료

```
x
```

### 테스트 실행

```
t
```
